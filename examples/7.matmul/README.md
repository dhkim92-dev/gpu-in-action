
# 행렬 곱

이 예제에서는 행렬곱을 GPU를 통해 계산하는 기초적인 커널을 두가지 제공합니다.

## 문제 정의


입력 행렬 $A$ (M x N)와 $B$ (N x K)가 주어졌을 때, 출력 행렬 $C$ (M x K)는 다음과 같이 계산됩니다.

$$
C_{i,j} = \sum_{k=1}^{N} A_{i,k} \cdot B_{k,j}
$$


## 커널 종류

### Naive 커널
각 행과 열에 대해 반복문을 사용하여 곱셈과 덧셈을 수행하는 가장 기본적인 방식입니다. 모든 스레드가 각 결과 행렬의 원소 $C_{i,j}$를 독립적으로 계산합니다. 메모리 접근이 비효율적입니다.

### Tiled 커널 (가초야)
행렬을 작은 타일(tile) 단위로 분할하여, 각 타일을 공유 메모리에 로드한 뒤 계산합니다. 이를 통해 글로벌 메모리 접근 횟수를 줄이고, 메모리 대역폭을 효율적으로 사용할 수 있습니다. 일반적으로 성능이 더 우수합니다.
